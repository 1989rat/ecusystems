1) Наименование калибровки
2) Размерность
3) Имя переменной для ссылок (возможно еще как-то используется, но точно применяется для ссылок в таблицах)
4) Ось Х (имя переменной - ссылка на калибровку оси Х)
5) Ось У (имя переменной - ссылка на калибровку оси У)
6) Вспомогательная переменная для визуализации 1
7) Вспомогательная переменная для визуализации 2
8) Описание калибровки (непосредственно формат)
9) Значение калибровки

Теперь формат описания калибровки:
[Адрес] [размерность] [множитель] [делитель] [смещение] [точность (знак после запятой)] [тип калибровки] [столбцы] [строки] [адрес бита если калибровка битовая][резерв] [резерв] [перевернутая ось х] [перевернутая ось у][min][max]

формула расчета такая: v = значение * множитель / делитель + смещение

Размерность:
1 – char
2 – unsigned char
3 – int
4 – unsigned int
5 – bit

Типы калибровок: 
1 - const 
2 - var
3 - table
5 - teach


Протокол обмена очень прост:
Скорость 9600,
Чтение по адресу:
‘r’ <addr1> <addr2> в ответ один байт данных
Запись по адресу:
‘w’ <addr1> <addr2> <data>
